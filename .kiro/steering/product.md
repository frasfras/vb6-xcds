# Product Overview

This is a Visual Basic 6 (VB6) designer clone built as a web application. It recreates the classic VB6 IDE experience with a drag-and-drop form designer, toolbox, properties panel, and code window.

## Key Features

- Drag-and-drop form designer with resizable controls (Button, Label, TextBox)
- Floating windows (Toolbox, Properties, Code Window, Form Designer)
- VB6-style event handlers (Click, Change, KeyPress)
- Save/load form layouts as JSON
- Generate React95 JSX code from designed forms
- Run mode to preview forms in action
- Windows 95 aesthetic using react95 UI library

## User Experience

The application mimics the classic VB6 development environment with a nostalgic Windows 95 look and feel. Users can design forms visually and export them as React components.
